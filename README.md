# PVLM YouTube Downloader

Download YouTube videos and music easily.

## Options

### 1. Web Version (Local)
- Run `python app.py`
- Open http://localhost:5000
- Works perfectly on your computer

### 2. Desktop App (EXE)
- Run `cd electron && npm install && npm run build`
- Find EXE in `electron/dist/`
- Install on any Windows PC
- Works offline!

### 3. Mobile App (APK)
- Go to https://www.pwabuilder.com
- Enter your Railway URL (if working)
- Download APK for Android

---

## Quick Start (Local)

```bash
# Install dependencies
pip install -r requirements.txt

# Run the app
python app.py

# Open browser
http://localhost:5000
```

---

## Files
- `app.py` - Main server
- `templates/index.html` - Web UI
- `static/` - Logo, PWA files
- `electron/` - Desktop app files
- `requirements.txt` - Python dependencies
